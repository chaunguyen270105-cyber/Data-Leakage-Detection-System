# Scripts Hỗ trợ

Thư mục này chứa các script tự động hóa để triển khai và thu thập dữ liệu trên các thiết bị endpoint.

## Danh sách
- `install.sh`: Tự động cài đặt các thư viện cần thiết (`requests`, `watchdog`, `psutil`) trên môi trường Linux/Wazuh Server.
- `clipboard.ps1`: Script PowerShell chạy trên máy Windows để ghi lại nội dung clipboard người dùng vào file log, phục vụ việc giám sát dữ liệu nhạy cảm.
